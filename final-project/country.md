# 4. Output and Interpretation 
- Fréquence : Les TX90p sont plus fréquents dans les zones désertiques (Sahara central, Libye, Égypte).
- Distribution : Les régions côtières (Maroc, Tunisie) montrent une fréquence plus modérée grâce à l’influence maritime.
- Tendance : Les anomalies saisonnières révèlent une intensification des extrêmes en été, particulièrement dans les zones intérieures.
